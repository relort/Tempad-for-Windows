Tempad - Temporary Note Editor. Notepad alternative.

Idea, implementation by Pavel Bugaevskiy.


— — — — 
Product developed in Delphi 2010.

Folder struct:
	- Deps   - various libs and components used in Tempad
	- Source - source (thx to Captain Obvious)

You should write your own paths to Deps in Project Settings to compile Tempad.
